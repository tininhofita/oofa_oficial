# Referência do Sistema de Cores

> Princípios de teoria das cores, processo de seleção e diretrizes para tomada de decisão.
> **Não memorize hex codes - aprenda a PENSAR sobre a cor.**

---

## 1. Regra 60-30-10

### Princípio de Distribuição

```
60% PRIMÁRIA (Fundo, grandes áreas)
→ Deve ser neutra ou calma.
→ Define o tom geral.

30% SECUNDÁRIA (Cards, seções, cabeçalhos)
→ Suporta sem dominar.

10% ACENTO (CTAs, destaques, atenção)
→ Atrai o olhar para o objetivo.
```

## 2. Psicologia das Cores no Contexto

| Se o projeto é...         | Considere estas cores  | Por quê?                        |
| ------------------------- | ---------------------- | ------------------------------- |
| **Finanças / Tecnologia** | Azuis, Cianos          | Confiança, estabilidade, calma. |
| **Saúde / Wellness**      | Verdes, tons terrosos  | Crescimento, saúde, orgânico.   |
| **Luxo / Premium**        | Preto, Ouro, Esmeralda | Sofisticação, elegância.        |
| **Urgência / Promoções**  | Vermelho, Laranja      | Ação, atenção, energia.         |

> ⚠️ **BANIMENTO DE ROXO:** IA tende a usar roxo demais. Use Verde Esmeralda, Azul Marinho ou Tons Terrosos no lugar.

## 3. Geração de Paletas (Método HSL)

Use **HSL (Hue, Saturation, Lightness)** para maior controle:

- **Hue (0-360):** A cor em si.
- **Saturation (0-100%):** Intensidade. Baixa = Sofisticado; Alta = Vibrante.
- **Lightness (0-100%):** Brilho. 50% é a cor pura.

### Regras para Modo Escuro

1. **Nunca use preto puro (#000000):** Use um cinza muito escuro com leve matiz da cor da marca.
2. **Reduza a saturação:** Cores muito vibrantes cansam os olhos no escuro.
3. **Elevação = Brilho:** Elementos mais "altos" na UI devem ser ligeiramente mais claros.

## 4. Checklist de Cores

- [ ] **Acessibilidade:** Contraste verificado via WCAG? (Mínimo 4.5:1 para texto normal).
- [ ] **Contexto:** A cor combina com a indústria e o público?
- [ ] **Consistência:** A regra 60-30-10 foi seguida?
- [ ] **Diferenciação:** Esta paleta é diferente do meu "porto seguro" usual?
